# DOM Manipulation Quiz Project

## Overview
This project is a DOM Manipulation Quiz created using **HTML, CSS, and JavaScript**.  
The quiz tests users on their knowledge of the **Document Object Model (DOM)** in three categories:

1. **DOM Basics**  
2. **DOM Manipulation**  
3. **DOM Events**

Users select answers for 9 questions (3 per category) and click the "Submit Quiz" button to see their **score as an alert**.

---

## Features
- Interactive quiz using **DOM events**.
- Score calculation implemented with **JavaScript functions**.
- **Responsive and clean styling** with CSS.
- Categorized quiz questions with proper headings.
- Real-time feedback via browser alert.
